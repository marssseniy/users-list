# Users List

Небольшое frontend-приложение для отображения и поиска пользователей.

## Demo

https://marssseniy.github.io/users-list/

---

## Технологии

- HTML5
- CSS3
- JavaScript
- Fetch API

---

## Функционал

- загрузка пользователей с сервера
- поиск пользователей по имени
- обработка состояния загрузки
- обработка ошибок соединения
- отображение сообщения при отсутствии совпадений
- адаптивный интерфейс

---

- работе с Fetch API
